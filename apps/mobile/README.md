# @vneclassno/mobile

Expo / React Native клиент для iOS/Android.

## Команды

- `npm run -w @vneclassno/mobile dev`
- `npm run -w @vneclassno/mobile ios`
- `npm run -w @vneclassno/mobile android`

## Текущее состояние

- Инициализирован app shell с tab navigation.
- Подключены Query client и базовый state слой.
- Подготовлен базовый API client для интеграции с backend.
